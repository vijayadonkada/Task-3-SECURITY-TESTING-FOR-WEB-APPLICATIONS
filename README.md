# Task-3-SECURITY-TESTING-FOR-WEB-APPLICATIONS

## Project Overview
This project demonstrates security testing performed on a sample vulnerable web application (DVWA) to identify common web vulnerabilities.

## Scope
- Application: Damn Vulnerable Web Application (DVWA)
- Environment: Localhost using XAMPP

## Vulnerabilities Identified
- SQL Injection
- Reflected Cross-Site Scripting (XSS)
- Stored Cross-Site Scripting (XSS)

## Tools Used
- XAMPP
- DVWA
- Web Browser

## Outcome
The application was found to be vulnerable to multiple high and medium severity issues. Mitigation strategies were proposed to improve security.
